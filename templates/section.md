# Standard section

```
      <section id="sectionId">
        <a name="sectionId"></a>
        <h4><a href="#sectionId">Section Title</a></h4>
        <p>Content</p>
      </section>
```

## Section with overlay

```
      <section id="sectionId">
        <a name="sectionId"></a>
        <div>
          <!-- you can add background content here -->
          <div class="overlay">
            <h4><a href="#sectionId">Section Title</a></h4>
            <p>Content</p>
          </div>
        </div>
      </section>
```
